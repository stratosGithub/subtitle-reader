# SubReader

A mobile-friendly SRT subtitle reader that works as a Progressive Web App (PWA).

## Quick Start

```bash
cd subtitle-reader
python3 serve.py
```

Then open the **Network URL** (e.g. `http://192.168.x.x:8080`) on your iPhone while connected to the same Wi-Fi network.

## Install on iPhone

1. Open the URL in Safari
2. Tap the **Share** button (square with arrow)
3. Tap **Add to Home Screen**
4. The app now works like a native app, even offline

## Features

- Load `.srt` subtitle files
- Dark theme optimized for mobile
- Table view with start/end times and text
- Reader view for comfortable reading
- Search/filter subtitles
- Tap any subtitle to see it full-screen with swipe navigation
- Select multiple subtitles
- Works offline after first load (PWA)
